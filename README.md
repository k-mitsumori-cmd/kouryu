# 交流会タスク管理

交流会を主催・開催する際のタスクを洗い出し、スケジュールを自動生成するツールです。

## 特徴

- 🚀 **簡単操作**: イベント情報を入力するだけでタスクリストを生成
- 🎯 **お試し機能**: 初めての方でもサンプルデータで動作確認可能
- 🤖 **AI生成**: OpenAI APIを使用して高品質なタスクリストを自動生成
- 📅 **スケジュール表示**: タイムライン形式でスケジュールを視覚化
- 📊 **状態管理**: タスクの状態（未着手、進行中、完了）を管理
- 📋 **CSV出力**: タスクリストをCSV形式でダウンロード可能

## セットアップ

### 1. ローカル開発環境での動作確認

`index.html`をブラウザで開くだけです！

- **ダブルクリック**: `index.html` をダブルクリックしてブラウザで開く
- **ドラッグ&ドロップ**: `index.html` をブラウザにドラッグ&ドロップ

**注意**: AI機能を使用する場合は、Vercelにデプロイするか、ローカルでサーバーを起動してください。

### 2. Vercelでのデプロイ（推奨）

#### ステップ1: GitHubにプッシュ

```bash
cd kouryu
git init
git add .
git commit -m "Initial commit"
git remote add origin https://github.com/k-mitsumori-cmd/kouryu.git
git branch -M main
git push -u origin main
```

#### ステップ2: Vercelでデプロイ

1. https://vercel.com にアクセス
2. GitHubアカウントでログイン
3. 「Add New...」→「Project」をクリック
4. `k-mitsumori-cmd/kouryu` リポジトリをインポート
5. 環境変数を設定：
   - 変数名: `OPENAI_API_KEY`
   - 値: あなたのOpenAI APIキー
6. 「Deploy」ボタンをクリック

## 使い方

1. **お試しボタン**: 初めて使う方は「お試しボタン」をクリックしてサンプルデータを読み込み
2. **イベント情報入力**: 
   - イベント名を入力
   - 開催日を選択
   - イベント種別を選択
   - イベント詳細を入力（規模、会場、特別な要件など）
3. **タスクリスト生成**: 「タスクリストを生成する」ボタンをクリック
4. **タスク管理**: 
   - 各タスクの状態をクリックして変更（未着手 → 進行中 → 完了）
   - タスクを確認し、必要に応じて編集
5. **CSV出力**: 「CSV出力」ボタンでタスクリストをダウンロード
6. **再生成**: 「再生成」ボタンで異なるタスクリストを生成

## 生成されるタスクカテゴリ

1. **集客・広報**
   - メール文案作成
   - 招待状送付
   - リマインド通知配信
   - 既存への声かけ
   - など

2. **当日運営準備**
   - 受付フロー設計＆台本作成
   - 名札・備品搬入リスト作成
   - 受付・誘導担当の役割分担
   - 音響／マイクチェック
   - プログラム進行リハーサル
   - など

3. **イベント実行**
   - 受付開始
   - 開会挨拶・趣旨説明
   - 登壇者ピッチ
   - 交流タイム
   - クロージング・次回告知
   - 片付け・撤収
   - など

4. **フォローアップ**
   - 参加者アンケート配信
   - 獲得アポ件数の集計
   - リードフォローリスト作成
   - 成果報告レポート作成
   - 次回振り返りMTG設定
   - など

## 技術スタック
